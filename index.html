HTML

HTML stands for: Hyper Text Markup Language

This is the code to markup the content of a web page.

h1 Hello h1
  • You can add or subtract content from a webpage
  • You do this using a plain text editor, there are many available to use!
  • HTML content consists of an opening tag, the content and a closing tag.
Semantic Tags

Semantic Tags are used to make up the structure of the webpage.

They make is possible markup sections of the content based on their purpose/ functionality.

h1, p, div, nav
  • h1 is used for creating main headings.
  • p stands for paragraph, its used for writing text.
  • div is used for putting content in a "box"it can contain mnay different other semantic tags.
CSS

CSS stands for Cascading style sheets.

Css provides style and layouts to a webpage. It is very helpful in creating a webpage that looks nice!

body{ color: blue; }
  • Can be used to change the color of text, backgorunds, boxes etc.
  • Can be used for creating a specific layout for the page. Can help rearrange content
  • It can also be used to size text, make it bigger or smaller.

Media Queries

Media Queries are used to define different layouts for different browser sizes.

Breakpoints are used for this. A breakpoint is the point at which a website's content will change a certian way in order to give the user the best view. Ther are common breakpoints:

@media(max-width:480px){}
  • 480px for mobile phones
  • 768px for tablets
  • 1024px for desktops
